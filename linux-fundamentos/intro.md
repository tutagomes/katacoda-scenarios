## Bem-vindo ao Linux!

Se você nunca abriu um terminal antes, este é o lugar certo para começar.

O **Linux** é o sistema operacional que roda na grande maioria dos servidores, containers Docker, sistemas em nuvem e dispositivos embarcados do mundo. Aprender Linux não é opcional para quem trabalha com tecnologia — é uma habilidade fundamental.

### O que é um terminal?

O terminal é uma janela onde você digita comandos de texto para o computador executar. Parece intimidador no início, mas depois que você pega o jeito, é muito mais rápido e preciso do que usar uma interface gráfica.

O **shell** é o programa que interpreta os comandos que você digita. O mais comum no Linux é o **Bash** (Bourne Again Shell).

```
arthur@servidor:~$  ← isso é o prompt
  │       │    │
  │       │    └── ~ significa que você está na sua pasta pessoal (home)
  │       └─────── nome da máquina
  └─────────────── seu usuário
```

### O que você vai aprender neste cenário

- Entender o prompt e os primeiros comandos
- Navegar pela estrutura de pastas do Linux
- Criar, copiar, mover e remover arquivos e pastas
- Ler o conteúdo de arquivos no terminal
- Editar arquivos de texto com o editor `nano`
